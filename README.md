# 🎣 Restaurant Kayan

A small **math + fishing** web game built with **Svelte + Vite**. Everything lives in one continuous world: a **pond** on the left, a **math sign** in the middle, and **Restaurant Kayan** on the right — all on a comfortable green field. Works on desktop, tablet, and mobile.

**▶ Play:** https://grigorysolomatov.github.io/restaurant-kayan/

### Game loop
1. Solve the math problem on the sign in the middle (e.g. `2 + 2`).
2. A correct answer lets you **catch the fish** swimming in the pond — tap it.
3. The fish flies to **Restaurant Kayan** and is sold for money 💰.

### Develop
```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
```

Deployed to GitHub Pages from the `gh-pages` branch (the built `dist/`).
